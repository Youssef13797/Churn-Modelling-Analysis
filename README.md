# 📊 Bank Customer Churn Analysis

## 📌 Project Overview
This project focuses on analyzing customer data to understand the factors behind customer churn in the banking sector. Using **Python**, I performed a deep Exploratory Data Analysis (EDA) to uncover trends related to geography, gender, and credit scores.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Plotly Express (for interactive charts).
- **Environment:** Jupyter Notebook.

## 🔍 Key Highlights
- **Data Auditing:** Cleaned the dataset and ensured zero null values.
- **Geographic Insights:** Identified that France holds ~50% of the customer base.
- **Interactive Visuals:** Created dynamic Histograms and Heatmaps to visualize correlations between demographics and churn.

## 📂 Files in this Repo
- `Churn_Modeling_Analysis.ipynb`: The full analysis and visualization code.
- `Churn_Modelling.csv`: The raw dataset used for the project.
